# Changelog

## 0.1.0

- Initial release
- Canvas-based TMX map rendering
- Layer visibility controls
- Zoom, pan, and fit-to-window
- Grid overlay
- Tile coordinate and GID inspection on hover
- Auto-reload on file change
- Tileset image auto-discovery with extension fallback
- Tile flip support (horizontal, vertical, diagonal)
- Colored fallback rendering for missing tilesets
